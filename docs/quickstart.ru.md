# Stream.Genome простыми словами

## Публичная часть

Публичный пользователь открывает:

```text
http://localhost:4000
```

Он видит портал знаний: мемы, связи, источники, timeline, свежие фрагменты и общую карту лора.

Публичный пользователь не должен запускать сканирование, добавлять каналы или вручную загружать данные.

## Админская часть

Операционные действия живут отдельно:

```text
http://localhost:4000/admin
```

Там находятся:

- регистрация YouTube-канала;
- ручная вставка тестового фрагмента;
- будущие фоновые сканеры;
- будущие очереди и расписания;
- будущая модерация источников.

В production админская зона требует переменные:

```text
STREAM_GENOME_ADMIN_USER
STREAM_GENOME_ADMIN_PASSWORD
```

## Зарегистрировать YouTube-канал

Админ открывает:

```text
http://localhost:4000/admin/youtube/channel
```

И вставляет адрес:

```text
https://www.youtube.com/@channel
```

Пока это только сохраняет канал как реальный источник. Видео и субтитры еще не скачиваются.

После регистрации откроется страница источника. Там можно нажать:

```text
Start Scan
```

Сейчас scan создает запись запуска и ставит фоновую задачу. Реальное получение списка видео будет следующим этапом.

## Вставить тестовый фрагмент

Админ открывает:

```text
http://localhost:4000/admin/ingest
```

И вставляет несколько строк:

```text
Streamer: Again with the chair arc?
Chat: CHAIR ARC RETURNS
Mira: The chair arc never died.
```

Это ручной тестовый вход. Он нужен для разработки, а не для обычных пользователей.

## Главная идея

Публичный портал:

```text
смотреть, искать, изучать, возвращаться
```

Админская зона:

```text
подключать источники, запускать обработку, управлять системой
```
