# Roadmap

## Этап 1: Lore Ledger

- Сохранение источников, content items, сегментов, сущностей, алиасов, observations, событий, связей и записей эволюции мемов.
- Постановка ingestion и intelligence-задач в очередь через Oban.
- Добавление fixture-based тестов для ingestion.
- Добавление представлений source evidence для каждого сгенерированного lore-объекта.

## Этап 2: Extractors

- Реализация парсеров транскриптов, чатов, комментариев и экспортов сообществ.
- Добавление entity extraction через AI-провайдеров.
- Добавление извлечения связей и обнаружения колбэков.
- Сохранение confidence score, метаданных модели и версии prompt’а для каждого сгенерированного evidence-объекта.

## Этап 3: Графовая база данных

- Выбор Neo4j или Memgraph в качестве первого production graph adapter.
- Синхронизация изменений сущностей и связей.
- Добавление graph traversal-запросов для колбэков, конфликтов, происхождения мемов и долгоживущих сюжетных арок.

## Этап 4: Timeline Intelligence

- Построение chronology replay-запросов.
- Отслеживание состояний жизненного цикла мемов:
  - emerging
  - growing
  - stable
  - declining
  - dormant
  - revived
  - dead
- Добавление rollup-аналитики эмоциональной интенсивности по временным периодам, источникам, стримам и сообществам.

## Этап 5: Creator Universe UI

- Интерактивная universe map.
- Представление семейного дерева мемов.
- Режим воспроизведения лора.
- Эмоциональная heatmap.
- Поиск по lore memory на естественном языке.

## Этап 6: Продвинутые narrative-возможности

- Documentary mode.
- Semantic clip retrieval.
- Обнаружение audience factions.
- Предсказание мемов.
- Обнаружение narrative collapse.
